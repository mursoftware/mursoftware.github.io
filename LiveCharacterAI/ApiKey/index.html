OpenAI APIキー取得方法

OpenAIアカウント作成

OpenAI developer platform

上記サイトよりサインアップ


メールアドレスまたはGoogleアカウントなどで続ける


APIを選択（ChatGPTを使用していない場合、この画面は表示されない）


クレジットカード登録

※＄5の無料枠があるが、利用するためにはクレジットカードの登録が必要


Settings（設定）を押す


Billing（請求）を押す


Add payment details（支払詳細を追加）を押す


Individual（個人）を選択


クレジットカード情報を入力しContinueを押す


APIキーの発行

API keys（APIキー）を押す


Start verification（認証を開始する）を押す


電話番号を入力しSend codeを押す

電話に届いたコードを入力する


APIキーに適当な名前を入力し、Create secret key（シークレットキーを作成）を押す


発行されたAPIキーをコピーし保存する
※このコードは後で表示することができないためここで保存すること
